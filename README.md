# openai-openapi-json

OpenAI OpenAPI spec (`manual_spec` branch) auto-converted to JSON.

Source: https://github.com/openai/openai-openapi/tree/manual_spec

## Direct download

```
https://raw.githubusercontent.com/clsandoval/openai-openapi-json/main/openai-openapi.json
```

## jsDelivr CDN

```
https://cdn.jsdelivr.net/gh/clsandoval/openai-openapi-json@main/openai-openapi.json
```

The JSON is regenerated weekly and on every push via GitHub Actions.
